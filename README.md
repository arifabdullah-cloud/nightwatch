# Nightwatch

> Production-grade cloud reliability platform for Azure.

Nightwatch is a portfolio project that demonstrates how modern cloud platforms are designed, deployed, monitored, and operated.

The project focuses on Cloud Engineering, Site Reliability Engineering (SRE), Platform Engineering, Infrastructure as Code, Observability, and Automation.

## Vision

Build a production-like cloud environment that can:

- Monitor infrastructure health
- Detect incidents
- Diagnose failures
- Recommend corrective actions
- Perform automated remediation
- Simulate realistic workloads

## Planned Architecture

- Azure
- FastAPI
- PostgreSQL
- Redis
- Azure Monitor
- Azure Container Apps
- GitHub Actions
- Bicep
- Docker
