# Nightwatch

> Production-grade Cloud Reliability Platform for Azure.

Nightwatch is a hands-on portfolio project that demonstrates the design, deployment, and operation of modern cloud infrastructure using Azure and open-source technologies.

Unlike traditional CRUD applications, Nightwatch focuses on the operational side of software engineering—building a production-like environment that can be monitored, stressed, diagnosed, and continuously improved through automation.

The goal is to simulate how Cloud Engineers, Site Reliability Engineers (SREs), and Platform Engineers operate real-world systems in production.

---

## Vision

Nightwatch aims to provide a realistic cloud environment capable of:

- Discovering cloud resources
- Monitoring infrastructure and application health
- Collecting logs and metrics
- Detecting operational issues
- Performing incident analysis
- Generating operational recommendations
- Supporting automated remediation
- Simulating production workloads and failure scenarios

This project prioritizes engineering practices over business functionality.
